# 🎥 AI YouTube Video Summarizer

An AI-powered application that summarizes YouTube videos using transcript extraction and Google's Gemini model.  
The system automatically fetches the video transcript and generates a concise summary using AI agents.

---

## 🚀 Features

- Extracts transcript from any YouTube video
- AI-powered summary generation
- Uses CrewAI agents for structured workflow
- Clean Streamlit frontend
- Gemini LLM integration
- Environment variable support for API keys

---

## 🧠 How It Works

1. User enters a YouTube video URL.
2. The system extracts the video ID.
3. Transcript is fetched using the YouTube Transcript API.
4. CrewAI agents process the transcript.
5. Gemini generates an AI-powered summary.
6. The summary is displayed in the Streamlit interface.
